The functionality of this script (xbox_ban_chat.py being main.py) was to monitor/scan the Xbox Official Network banlist for Ark Survival Evolved, and invite all new additions into an Xbox group chat with an image of the "Globally Banned" error message popup when you are banned and tried joining an official server.
